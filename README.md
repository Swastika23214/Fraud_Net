# Fraudulent Telephonic Pattern Detection System
The project aims to: Create a fraud detection system based on clear, set rules. Use data structures to store call data for quick access. Spot suspicious calls that happen often and don't last long. Find fraud groups by using graph methods, detecting patterns. Use blacklists, check prefixes to catch scams. Make reports of suspicious numbers.

## 🚀 Features
- CDR Data Ingestion  
- Rule-Based Fraud Detection  
- Graph-Based Fraud Ring Detection (BFS/DFS)  
- Sorting & Ranking (Merge Sort)  
- Report UI Dashboard  

## 🛠️ Tech Stack
- C++  
- HTML, CSS, JavaScript  
- SQLite
